# 🚀 MVP 콘텐츠 기획안: 산업별 프롬프트 라이브러리 (Industry Prompt Library)

## 1. 프로젝트 개요 (Executive Summary)
**프로젝트명:** PromptCraft Industries  
**비전:** "산업별 최적화된 프롬프트를 통해 사용자의 업무 효율을 즉시 높이는 도구"  
**MVP 목표:** 3 가지 핵심 산업군(마케팅, 개발/기술, 교육)에 대한 고수익 프롬프트 팩 출시 및 첫 수익화 흐름 검증.

### 핵심 가치 제안 (Value Proposition)
> "단순히 단어를 나열한 프롬프트가 아닙니다. **실제 현장에서 검증된 결과물**을 만들어내는 '작업 절차'와 '맥락'이 담긴 라이브러리입니다."  
- **근본적 욕구 해결:** 초보자가 전문가처럼 결과를 낼 수 있도록 도와줌 (소속감·성취감)
- **시간 절약:** 반복적인 프롬프트 구성 없이 바로 실행 가능

---

## 2. 타겟 청중 및 시장 분석
**주요 타겟:**  
1.  **초급~중급 프리랜서/소상공인:** 업무 효율성을 높이고 수익을 빠르게 창출하려는 자.  
2.  **기업 내 AI 도입 담당자:** 팀원들의 생산성 증대 도구로 구매 가능한 솔루션을 찾는 자.  

**현재 시장 니즈:**  
- LLM 의 발전으로 "어떤 프롬프트를 쓰면 좋은지"가 명확해졌으나, **맥락 없는 단편 지식**만 제공됨.  
- 산업별 특성에 맞는 **구조화된 프롬프트 템플릿**과 **사용 가이드**에 대한 수요 증가.

---

## 3. 콘텐츠 구조 및 전략
MVP 단계에서는 **1 개 산업군 (Deep Dive)** 으로 집중 후 확장하는 '초점 전략'을 취합니다.  
다음 3 가지 산업군 중 하나를 선택하여 첫 번째 콘텐츠로 출시:  

### [산업별 프롬프트 라이브러리] — 선택 옵션
| 산업 | 핵심 주제 | 타겟 고객 | 예상 수익화 모델 |
| :--- | :--- | :--- | :--- |
| **A. 마케팅/브랜딩** | SNS 기획, 광고 카피라이팅, 브랜드 톤 정의 | 마케팅 에이전트, 소규모 브랜드 | 프리미엄 프롬프트 팩 + 구독 (월 19,800 원) |
| **B. 개발/기술** | 코드 리뷰, 문서화 자동화, 버그 디버깅 | 백엔드/프론트엔드 개발자 | 기술 커뮤니티 연계 판매 + API 키 기반 |
| **C. 교육/상담** | 학습 계획 수립, 커리큘럼 설계, 멘토링 가이드 | 온라인 강사, 교육 기관 | 유료 강의 자료 병행 판매 (1 번 이상 구매 시 할인) |

> **MVP 전략:** 산업 A(마케팅/브랜딩) 를 선택하여 첫 번째 콘텐츠로 출시합니다.  
> - 이유: 타겟 청중의 돈 쓰기 심리가 가장 명확하고, Writer 에이전트가 카피라이팅 강점이 있어 직접 콘텐츠를 생성하기 용이함.

## 4. 콘텐츠 상세 기획 (Content Outline)
**콘텐츠 제목 예시:** "마케팅 프롬프트 라이브러리: 브랜드 가치를 10 배 높이는 5 단계 전략"  
- **섹션 1:** "당신의 브랜드가 가진 잠재력을 깨워주는 프롬프트" — 후크/인텐트 자극  
- **섹션 2:** "브랜드 톤을 일관되게 유지하는 자동화 워크플로우" — 근본적 욕구 해결  
- **섹션 3:** "실제 사례 분석: 어떤 단계를 거쳐 좋은 결과가 나왔는지" — 사회적 증거 (Social Proof)  
- **섹션 4:** "지금 바로 실행해보기" — CTA 유도

**각 산업별 프롬프트 팩 구성:**
- 총 50 개 이상의 프롬프트 (핵심 기능별로 분류)  
- 각 프롬프트에 대한 설명 및 사용 예시 포함  
- 사용자 맞춤형 결과물 생성을 위한 AI 워크플로우 가이드  

---

## 5. 수익화 모델 및 KPI
**수익화 방안:**
1.  **단일 프롬프트 팩 판매 (One-time purchase):** 산업별 기본팩 구매 시 즉시 다운로드 가능.  
2.  **구독 기반 업데이트 (Subscription):** 새로운 프롬프트 추가, AI 툴 사용 가이드 등 지속적 지원.  

**초기 KPI:**
- 30 일 내 첫 판매 목표: $500 (약 6 만 원)  
- 고객 획득 비용 (CAC) < $10 유지  
- 전환율 (Conversion Rate): 랜딩 페이지에서 2% 이상

## 6. 개발자에게 필요한 기능적 요구사항
코다리 (개발자) 에게 전달할 기술적 요구사항입니다:

### 필수 기능
- [ ] **프롬프트 데이터 관리:** 프롬프트 ID, 산업군, 태그, 생성일 등 메타데이터 저장 및 검색  
- [ ] **사용자 구매 기록 추적:** 구매 내역, 구독 상태, 라이선스 만료 일 관리  
- [ ] **API 호출 로깅:** 사용된 프롬프트별 성능 데이터 (성공/실패 비율) 수집  

### 추천 스택
- **프론트엔드:** React + Tailwind CSS  
- **백엔드:** Node.js + Express 또는 Python/FastAPI  
- **데이터베이스:** PostgreSQL 또는 Supabase (사용자 인증 및 결제 연동 용이성)  
- **결제 시스템:** Stripe 또는 PayPal (글로벌 고객 확보 시 권장)  

### 데이터 형식
프롬프트는 JSON 형태로 저장하여 쉽게 확장 가능하게 만듭니다:
```json
{
  "id": "prompt_001",
  "title": "브랜드 톤 정의 프롬프트",
  "industry": "marketing",
  "tags": ["branding", "tone"],
  "description": "사용자의 브랜드 정체성을 명확히 정의하는 프롬프트",
  "usage_example": "/** ... **/\n"
}
```
